# 参数计算顺序

```c++
void printSum(int a , int b)
{
	std::cout << a << " + " << b << " = " << a+b << std::endl; 
}

int main() {
	int value = 0;
	printSum(value++, value++);
	// 0 + 0 = 1;
}
```

在不同模式下结果不同。

这就是未定义行为，无法预测的行为

简单来说就是允许在编译是计算出变量的值，直接将value变为0

我们将c++标准调至17以上就好了
